= Neoland 

A new homeland for developing AI-native applications.

== Quick

[source, shell]

------
git clone https://github.com/zhenyuanlau/neoland.git

cd neoland
------

=== Docs

[source, shell]

make docs

=== Setup

[source, shell]

----
make

make ollama.llama3

make ollama.dolphincoder
----

=== Play

[source, shell]
------
make open
